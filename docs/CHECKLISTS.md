# Checklists

## Definition of Ready
- [ ] Requirement & acceptance criteria written
- [ ] Impact map (routes/modules/DB/flags)
- [ ] Risks & rollback defined
- [ ] Test plan (unit/integration/E2E)

## In-Progress
- [ ] Branch from latest `master` (rebased)
- [ ] Small cohesive commits
- [ ] Local type/lint/tests pass

## Definition of Done
- [ ] CI green (type/lint/tests)
- [ ] a11y & perf budgets verified (changed routes)
- [ ] Security checks green (SAST/secrets/deps)
- [ ] i18n/RTL verified (if UI touched)
- [ ] Telemetry added/updated
- [ ] README/ADR/Runbooks updated
- [ ] PR Impact Note present
